Aqui van las tablas

Para hacer una tabla se usa "table" la cual sera modificada con "border" para el borde de la tabla table border="1",
"tr" para abrir filas, dentro un "thead" para especificar cual es el encabezado, seguido abriremos un "tbody" para especificar el cuerpo
usaremos "tr" para abrir fila y "td" para cada valor que queremos añadir en orden de las columnas

Aqui va el formulario

Para hacer un formulario usaremos el comando "form", dentro añadiremos inputs, hay muchos tipos distintos
